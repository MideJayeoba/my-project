# Data Collection Protocol (IRB-Aligned)

## Scope
Collect consented, de-identified voice samples from Akure PHCs to improve ASR for medical Nigerian-accented and code-switched speech.

## Protocol Summary
1. Explain study purpose in participant's preferred language.
2. Obtain verbal and documented consent before recording.
3. Record symptom narratives and guided prompts in controlled sessions.
4. Remove direct identifiers from metadata before model training.
5. Store raw data only on approved local encrypted storage.

## Safety and ethics controls
- No transmission of patient recordings to external cloud services.
- Access controls for research team only.
- Audit trail for data access and preprocessing events.
