# User Trust Interview Guide (Akure PHC Pilot)

## Objective
Assess whether low-literacy users trust and understand spoken guidance from the assistant.

## Interview Prompts
1. Did the spoken response sound clear in your preferred language mix?
2. Did you feel the response matched your symptom description?
3. Would you follow this guidance before speaking to a nurse?
4. Did the assistant make you feel more confident or more worried?
5. What changes would make the voice assistant easier to use?

## Observation Checklist
- Time-to-first-response perception
- User confidence during microphone interaction
- Need for repeated prompts
- Confusion points in voice-only flow
